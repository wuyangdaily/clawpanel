三步 开始开发

Node.js 18+ & Rust stable & Tauri v2 依赖

terminal
# 克隆 & 安装
git clone https://github.com/qingchencloud/clawpanel.git
cd clawpanel && npm install
# 启动开发（完整 Tauri 桌面应用）
cargo tauri dev
# 或仅启动前端（浏览器调试）
npm run dev
# 构建发布
cargo tauri build
